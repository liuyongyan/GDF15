Comparison vs hand-written:
- hand rows: 704; hand unique genes: 686
- real is ~28x larger by rows
- gene-set Jaccard: 0.032; overlap genes: 626 (91% of hand-written gene set is present in real)

Class derivation: deterministic, first-match-wins ruleset over lowercased UniProt
keywords plus subcellular-location "Secreted" string. See `etl_uniprot.py::CLASS_RULES`.

Red flags / caveats:
- "unknown" bucket (4,608 records / 24%) captures proteins whose UniProt keywords match none of the rules — typically uncharacterized ORFs, ribosomal/spliceosome components without "enzyme" keywords, etc. Spec'd class list intentionally narrow; we do not invent new classes.
- For proteins with BOTH "Cytokine" and "Hormone" keywords (e.g., the canonical secreted growth-differentiation factor that the pipeline targets), the rule order places `secreted_hormone` ahead of `secreted_cytokine`. The hand-written snapshot labeled the same protein `secreted_growth_factor` (which would require the "Growth factor" keyword that the real UniProt entry does NOT carry). The real label is closer to the source-of-truth annotation.
